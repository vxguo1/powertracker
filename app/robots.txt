# powertracker.io
#
# Public, open-data project. The goal is visibility - both for human
# search and for LLM training / answer-engine retrieval. Every major
# web crawler and AI agent is explicitly welcomed below.
#
# Binary tile responses (/tiles/*.pmtiles) are NOT disallowed here so
# that Google's renderer and other JS-running agents can fetch them
# during render. They are instead marked X-Robots-Tag: noindex by the
# worker, which prevents them being indexed as standalone documents
# without blocking the render.
