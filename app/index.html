OD uptick (state, Z-score)

·normal (z < 1.5) ·elevated (1.5 ≤ z < 2) ·significant (2 ≤ z < 3) ·severe (z ≥ 3, pulsing)
Source: CDC VSRR Provisional Drug Overdose Deaths (state, monthly, trailing 12-month). Z computed per drug class (all-cause, fentanyl, heroin, methadone, semi-synthetic opioids, cocaine, psychostimulants) against baseline [t-72, t-12] months; state level = the most severe class that *also* persisted the prior month. Adapted from the spec's weekly cadence — NEMSIS week-resolution data is not public.

Homicide uptick (state, Z-score)

·normal (z < 1.5) ·elevated (1.5 ≤ z < 2) ·significant (2 ≤ z < 3) ·severe (z ≥ 3, pulsing)
Source: CDC Mapping Injury, Overdose, and Violence — State (fpsi-y8tj, intent All_Homicide). Z = (TTM rate − mean(2019-2023)) / stdev(2019-2023). State-level annual cadence (TTM = trailing 12 months). With only five baseline years the stdev estimate is noisy — read Z as directional.

Power outage uptick (state, Z-score)

·normal (z < 1.5) ·elevated (1.5 ≤ z < 2) ·significant (2 ≤ z < 3) ·severe (z ≥ 3, pulsing)
Source: NOAA Storm Events Database. Per state, count outage-driving storm events (thunderstorm wind, tornado, ice storm, hurricane, high wind, winter storm, wildfire, lightning, flash flood) per month, then form a trailing-12-month sum. Z is against baseline [t-72, t-12] months (same shape as OD uptick). Persistence required. This is a weather-event proxy — federal customer-out data (DOE OE-417) is no longer published at a free public endpoint, so a state can score high if storm activity was unusual even if grids held up.
